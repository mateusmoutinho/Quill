

### Build Dependencies
#### [Darwin](https://github.com/OUIsolutions/Darwin)
For Build the project you must have [Darwin](https://github.com/OUIsolutions/Darwin) installed on version **0.20.0**
if you are on linux you can install darwin with:

```bash
curl -L https://github.com/OUIsolutions/Darwin/releases/download/0.20.0/darwin.c -o darwin.c &&
gcc darwin.c -o darwin.out &&
sudo mv darwin.out /usr/bin/darwin
```

### [Docker](https://www.docker.com/) 
For Build the project you must have [Docker](https://www.docker.com/) installed on version **24.0.7**
### [Podman](https://podman.io/)
For Build the project you must have [Podman](https://podman.io/) installed on version **4.0.1**
